---
name: Frontend Accessibility
description: Build accessible user interfaces with semantic HTML, keyboard navigation, ARIA attributes, and proper color contrast. Use this skill when creating UI components, adding form labels, implementing keyboard interactions, managing focus states, writing alt text for images, or ensuring screen reader compatibility. Apply when working with React components (.tsx, .jsx), HTML templates, form elements, modals, navigation menus, or any user-facing interface code.
---

## When to use this skill:

- When creating new UI components or pages
- When adding interactive elements (buttons, links, forms)
- When implementing keyboard navigation and focus management
- When writing alt text for images and icons
- When adding form labels and input descriptions
- When using ARIA attributes for complex components
- When structuring heading levels (h1-h6)
- When handling focus in modals and dialogs
- When ensuring sufficient color contrast
- When building navigation menus and skip links
- When working with .tsx, .jsx, or .html files
