## When to use this skill:

- When creating new database migration files
- When adding, removing, or altering database tables
- When creating or dropping indexes on tables
- When writing up/down migration methods
- When implementing rollback strategies
- When managing schema changes for zero-downtime deployments
- When separating schema migrations from data migrations
- When naming migration files descriptively
- When working with files in `/supabase/migrations/`, `/prisma/migrations/`
- When modifying .sql migration files
